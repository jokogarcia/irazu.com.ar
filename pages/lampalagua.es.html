Lampalagua

https://lampalagua.ar

Lampalagua es una editorial de libros boutique de La Rioja, Argentina.

Desarrollé su plataforma web utilizando Angular.js para el frontend y .NET con C# para el backend. El sitio cuenta con un CMS personalizado para gestionar listados de libros, información de autores y enlaces de compra de proveedores externos.

El diseño de la experiencia de usuario tiene un fuerte enfoque en la SIMPLICIDAD, esta es la característica que vendió el proyecto al cliente sobre otras soluciones enlatadas, como Shopify. Tiene exactamente las características que quieren, y nada más. Sin complejidad innecesaria, la plataforma es fácil de usar tanto para clientes como para administradores.

Esta es la razón, por ejemplo, por la que no hay función de carrito de compras, optando en su lugar por enlaces de compra externos. Esto fue solicitado por el cliente, para evitar competir con sus clientes directos (librerías y vendedores minoristas)

Características

Frontend

  • Diseño responsivo para una visualización óptima en todos los dispositivos
  • CMS personalizado para una fácil gestión de listados de libros e información de autores
  • Fuerte enfoque en la simplicidad y usabilidad, adaptado a las necesidades del cliente
  • Editor de texto enriquecido para la creación y edición de contenido
  • Integración con proveedores externos para enlaces de compra fluidos
  • Optimización SEO para mejorar la visibilidad online
  • Editor de texto enriquecido y gestión de imágenes

Backend

  • API robusta construida con .NET y C# para una gestión de datos fluida
  • SQLite para una gestión ligera de la base de datos
  • Integración con Auth0 para autenticación y autorización de usuarios administradores
  • Modelos de datos personalizables para una gestión de contenido flexible
  • Despliegue en contenedores usando Docker para una fácil escalabilidad
  • Despliegue seguro, sin dependencias, automatizado usando bash y ssh
    • Características de seguridad
    • Control de acceso basado en roles para usuarios administradores
    • Aplicación de CSP (Política de Seguridad de Contenido) para mitigar ataques XSS
