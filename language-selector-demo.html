Language Selector Component Demo

This page demonstrates the language selector web component in isolation.

The language selector automatically detects the current page and language from the URL.

It generates the appropriate links for the English, Spanish, and German versions of the current page.

The current language is automatically highlighted with bold text.

Testing

To test the component's language detection:

  • View this page as language-selector-demo.html (English)
  • View this page as language-selector-demo.es.html (Spanish)
  • View this page as language-selector-demo.de.html (German)

The component will automatically detect the language from the URL and highlight the appropriate language link.
